## About
Brain Tumor AI is a toolkit for brain tumor detection and segmentation from MRI images. Built for researchers and clinicians.

## Features
- Tumor detection and classification
- MRI image segmentation  
- Support for multiple tumor types
- Easy-to-use interface

## Installation
```bash
pip install brainsight
```  <-- ADD THESE 3 BACKTICKS HERE

## Model Zoo
Pre-trained models available for:
- Tumor segmentation (U-Net)
- Tumor classification (EfficientNet)

## Dataset
Works with BraTS dataset and custom MRI data.

## Requirements
- Python 3.8+
- TensorFlow/PyTorch
- 8GB RAM
